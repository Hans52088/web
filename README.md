# Custom Keyboard Visual Tools

這是一個專為自訂鍵盤開發所設計的網頁版視覺化工具集合。透過這些基於前端網頁技術的輕量級編輯器，開發者與愛好者可以更直覺地設計、預覽與調整鍵盤佈局。這些工具非常適合用來輔助開發 iOS 自訂鍵盤，或是針對 Hamster（落格輸入法）、大易（Dayi）及 Rime 輸入法框架進行佈局、滑動軌跡與外觀的客製化調整。

## 📦 專案內容與工具介紹

本儲存庫包含以下主要的網頁工具與視覺資源：

* **`index.html`**
    專案的主頁面與導覽入口，方便快速切換至不同的鍵盤編輯工具。

* **`keyboard_layout_color_tools.html`**
    **鍵盤佈局與配色工具**：提供視覺化的介面，讓你可以輕鬆配置鍵盤的按鍵排列與顏色主題，即時預覽各種主題風格下的鍵盤外觀。

* **`keyboard_swipe_hint_tools.html`**
    **滑動與長按 (Swipe & Hint) 編輯工具**：用來設定按鍵上的滑動或長按提示符號。在設計需要支援滑動與長按方式對應不同輸出的輸入法方案時，能提供相當直覺的設定介面。

* **`0.png` ~ `11.png`**
    一系列的圖片檔案，作為鍵盤佈局的預覽圖、設計參考範本、或是介面所需的圖形素材。
